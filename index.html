підвищення потенціалу
підготовки вчителів для відновлення
України

Project: 101237551 — EESF — ERASMUS-EDU-2025-CBHE

Мета та завдання проєкту

1.

Розробити навчальну дисципліну
"Освіта для сталого розвитку" та інтегрувати модулі зі питань сталого розвитку в існуючі навчальні дисципліни.

2.

Створити інноваційні навчальні матеріали з освіти для сталого розвитку.

3.

Підвищити професійну компетентність вчителів середніх шкіл через реалізацію структурованих програм підвищення кваліфікації у сфері сталого розвитку.

4.

Підвищити професійний рівень викладацького та адміністративного персоналу закладів вищої освіти шляхом проведення тренінгів із розвитку компетентностей у сфері сталого розвитку.

5.

Посилити співпрацю між українськими та європейськими освітніми установами через створення партнерств, проведення спільних тренінгів, досліджень і навчальних візитів.

6.

Сприяти цифровій трансформації у підготовці педагогів через розробку та впровадження онлайн-курсів з освіти для сталого розвитку.

7.

Забезпечити сталість та інституційну інтеграцію результатів проєкту.

8.

Розробити та впровадити (удосконалити) університетську стратегію гендерної рівності та провести семінари для академічного й адміністративного персоналу з даної тематики.

9.

Створити Університетські лабораторії сталого розвитку (SDG University Labs) — академічну платформу для інтеграції Цілей сталого розвитку у наукову, освітню та інноваційну діяльність університетів.
